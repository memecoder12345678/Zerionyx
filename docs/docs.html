Zerion Documentation & Libraries

Welcome to the official docs and standard library reference for Zerion.

Getting Started

  • Install Zerion: git clone https://github.com/memecoder12345678/zerion
  • Run a file: python ./interpreter/zerion.py yourfile.zer
  • Open interactive shell: python ./interpreter/zerion.py
  • See homepage for more info.

Recommended Environment

  • For learning: Python 3.11.9 or newer (best for beginners and educational purposes)
  • For automation: PyPy 3.11.11 (recommended when running repetitive tasks or automation scripts)
  • IDE recommendation: Use Zerion Editor with syntax highlighting for the best learning experience
  • Note: While the language is designed for simplicity, PyPy can significantly speed up automation tasks

Language Grammar

PROGRAM         ::= (STATEMENT NEWLINE*)*

STATEMENT          ::= SIMPLE_STATEMENT | COMPOUND_STATEMENT

SIMPLE_STATEMENT   ::=
      "load" STRING
    | "return" EXPR
    | "continue"
    | "break"
    | EXPR

COMPOUND_STATEMENT ::=
      IF_EXPR
    | FOR_EXPR
    | WHILE_EXPR
    | DEF_FUNC

EXPR               ::= 
      "let" IDENTIFIER "=" EXPR
    | IDENTIFIER "=" EXPR
    | IDENTIFIER ("+=" | "-=" | "*=" | "/=" | "//=" | "%=" | "^=") EXPR
    | COMP_EXPR (("and" | "or") COMP_EXPR)*

COMP_EXPR          ::= 
      ARITH_EXPR (("==" | "<" | ">" | "<=" | ">=" | "!=") ARITH_EXPR)*
    | "not" COMP_EXPR

ARITH_EXPR         ::= TERM (("+" | "-") TERM)*

TERM               ::= FACTOR (("*" | "/" | "//" | "%") FACTOR)*

FACTOR             ::= ("+" | "-") FACTOR | POWER

POWER              ::= CALL ("^" FACTOR)*

CALL               ::= ATOM ("(" (EXPR ("," EXPR)*)? ")")?

ATOM               ::= 
      INT | FLOAT | STRING | IDENTIFIER
    | "(" EXPR ")"
    | LIST_EXPR
    | LIST_INDEX
    | IF_EXPR
    | FOR_EXPR
    | WHILE_EXPR
    | DEF_FUNC
    | COMMENT

LIST_INDEX        ::= IDENTIFIER ">" EXPR

LIST_EXPR         ::= "[" (EXPR ("," EXPR)*)? "]"

IF_EXPR           ::= 
      "if" EXPR "do" STATEMENT
      (NEWLINE "elif" EXPR "do" STATEMENT)*
      (NEWLINE "else" STATEMENT)?
      NEWLINE "done"

FOR_EXPR          ::=
      "for" IDENTIFIER "=" EXPR "to" EXPR
      ("step" EXPR)?
      "do" STATEMENT
      NEWLINE "done"

WHILE_EXPR        ::= 
      "while" EXPR "do" STATEMENT
      NEWLINE "done"

DEF_FUNC          ::= 
      "defun" IDENTIFIER "(" (IDENTIFIER ("," IDENTIFIER)*)? ")"
      ("->" EXPR)?
      NEWLINE STATEMENT NEWLINE "done"

COMMENT           ::= "#" /[^\n]*/

Built-in Functions

  • append(obj, val) → list — Add val to end of obj return val
  • catch(func, args) → list — Run func with args return [result, none] if ok or [none, err] if error
  • clear() → none — Clear terminal
  • extend(a, b) → none — Add all in b to a both must be list
  • finally(f, a, final_f, final_a) → list — Run f with a then always run final_f with final_a return like catch
  • input(prompt) → string — Show prompt get user input
  • get_password(prompt) → string — Show prompt get user password
  • insert(lst, i, val) → none — insert val to lst at index i
  • is_func(x) → true/false — Check if x is func
  • is_list(x) → true/false — Check if x is list
  • is_none(x) → true/false — Check if x is none
  • is_num(x) → true/false — Check if x is number
  • is_str(x) → true/false — Check if x is string
  • is_thread(x) → true/false — Check if x is thread
  • len(x) → number — Get length of x
  • panic(msg, type) — Stop with err msg and type ('RT': 'Runtime Error', 'M': 'Math Error', 'IO': 'IO Error' or 'T': 'Type Error')
  • pop(lst, i) → any — Remove and return item at i in lst
  • print(text) → none — Prints text to the standard output with no newline
  • println(text) → none — Same as print but adds a newline
  • set_reusable(val, re) → list/num — Set reuse flag of val to re (true/false)
  • to_float(x, silent) → float/none — Convert x to float return none if fail and silent true else raise
  • to_int(x, silent) → int/none — Convert x to int return none if fail and silent true else raise
  • to_str(x) → string — Convert x to string
  • type(x) → string/none — Return type of x as string or none

Standard Libraries

list string math file hash memory net random sys threading time keyboard termcolor hashmap

list.zer

  • map(f, l) → list — Apply function f to each element of list l
  • filter(f, l) → list — Filter list l by function f (keep elements where f(x) is true)
  • reduce(f, l, i) → any — Reduce list l with function f and initial value i
  • min(l) → number — Minimum value in list l
  • max(l) → number — Maximum value in list l
  • list_slice(l, from, to) → list — Slice list l from from to to (exclusive)
  • reverse(l) → list — Reverse the list l
  • zip(l1, l2) → list — Pair elements from l1 and l2 (shortest length)
  • zip_longest(l1, l2) → list — Pair elements from l1 and l2 (fill with none if lengths differ)
  • change_value(l, i, v) → list — Change value at index i in list l to v
  • sort(l, reverse) → list — Sort list l (reverse: 0=asc, 1=desc)
  • count(l, v) → number — Count occurrences of v in l
  • index_of(l, v) → number/none — Index of v in l or none if not found
  • clear_list(l) → none — Remove all elements from l
  • list_multiply(l, n) → list — Repeat list l n times
  • rand_int_list(len, min, max) → list — List of random integers
  • rand_float_list(len, min, max) → list — List of random floats

  • join(sep, l) → str — Join list l with separator sep
  • replace(s, old, new) → str — Replace old with new in s
  • to_upper(s) → str — Uppercase s
  • to_lower(s) → str — Lowercase s
  • ord(s) → number — Unicode code point of character s
  • chr(n) → str — Character from code n
  • is_digit(s) → true/false — Is s a digit
  • is_ascii_lowercase(s) → true/false — Is s a lowercase letter
  • is_ascii_uppercase(s) → true/false — Is s an uppercase letter
  • is_ascii_letter(s) → true/false — Is s a letter
  • is_space(s) → true/false — Is s a whitespace character
  • find(s, v) → number/none — Index of substring v in s
  • find_all(s, v) → list — All indices of substring v in s
  • startswith(s, v) → true/false — s starts with v
  • endswith(s, v) → true/false — s ends with v
  • str_multiply(s, n) → str — Repeat string s n times

  • rand_int(min, max) → int — Random integer in [min, max]
  • rand_float(min, max) → float — Random float in [min, max]
  • rand_choice(list) → any — Random element from list
  • int_seed(i) → int — Integer seed

  • system(cmd) → none — Execute system command
  • osystem(cmd) → list — Same as system but return output
  • get_env(name) → str — Get environment variable
  • set_env(name, value) → none — Set environment variable

  • thread_is_alive(thread) → true/false — Is thread alive

  • sleep(seconds) → none — Sleep for seconds

keyboard.zer

  • keyboard_write(text) → none — Type text as keyboard input

  • keyboard_release(key) → none — Release a key
  • keyboard_wait(key) → none — Wait for key press
  • keyboard_is_pressed(key) → true/false — Is key pressed

termcolor.zer

  • cprint(text, color, bg, style) → none — Prints text with color, background, and style with no newline
  • cprintln(text, color, bg, style) → none — Same as cprint but adds a newline

hashmap.zer

  • hashmap(l1, l2, default) → list — Create a new hashmap (a list of [key, value] pairs) from l1 (keys) and l2 (values); missing values are filled with default
  • get_hashmap_value(hm, key, default) → any — Return the value for key in hm, or default if the key is not present
  • set_hashmap_value(hm, key, value) → none — Insert or update the entry for key in hm with value
  • has(hm, key) → true/false — Check if key exists in hm
  • del_key(hm, key) → none — Remove the entry for key from hm if it exists
  • get_hashmap_keys(hm) → list — Return a list of all keys in hm
  • get_hashmap_values(hm) → list — Return a list of all values in hm
  • get_hashmap_items(hm) → list — Return the raw list of [key, value] pairs in hm
  • clear_hashmap(hm) → none — Remove all entries from hm, leaving it empty

How to Use Libraries

  • To use a library, add load "libs.<name>" at the top of your .zer file.
  • Example: load "libs.math"
  • Then call functions as documented above.

Zerion Editor

About Zerion Editor

  • Modern, lightweight code editor designed for Zerion language
  • Syntax highlighting, auto-complete, file explorer, markdown preview
  • Open images, manage folders, and more
  • Cross-platform (Windows/Linux/Mac, requires Python 3.11+ and PyQt5)

Key Features

  • Tabbed editing, drag & drop file tree, and project explorer
  • Live Markdown preview for .md files
  • Quick commenting/uncommenting for Zerion code
  • Auto-completion for keywords, functions, and types
  • Image viewer for common formats (PNG, JPG, GIF, etc.)

Keyboard Shortcuts

  • Ctrl+N: New file
  • Ctrl+O: Open file
  • Ctrl+K: Open folder
  • Ctrl+Shift+K: Close folder
  • Ctrl+S: Save file
  • Ctrl+Shift+S: Save as
  • Ctrl+F: Find
  • Ctrl+H: Find and Replace
  • Ctrl+B: Toggle file tree
  • Ctrl+Shift+E: Show explorer
  • Ctrl+Q: Exit
  • Ctrl+/: Toggle comment (Zerion files)
  • Ctrl+Space: Autocomplete suggestions
  • Ctrl+P: Toggle Markdown preview
  • Ctrl+X: Cut line (when nothing selected)
  • Ctrl+Z / Ctrl+Y: Undo / Redo
  • Ctrl+A: Select all
To launch: python ./ide/zerion_editor.py
