Zerion Documentation & Libraries

Welcome to the official docs and standard library reference for Zerion.

Getting Started

  • Install Zerion: git clone https://github.com/memecoder12345678/zerion
  • Run a file: python zerion.py yourfile.zer
  • Open interactive shell: python zerion.py
  • See homepage for more info.

Recommended Environment

  • For learning: Python 3.11.9 or newer (best for beginners and educational purposes)
  • For automation: PyPy 3.11.11 (recommended when running repetitive tasks or automation scripts)
  • IDE recommendation: Use Zerion Editor with syntax highlighting for the best learning experience
  • Note: While the language is designed for simplicity, PyPy can significantly speed up automation tasks

Standard Libraries

list string math file hash memory net random sys threading time keyboard

list.zer

  • map(f, l) → List — Apply function f to each element of list l
  • filter(f, l) → List — Filter list l by function f (keep elements where f(x) is true)
  • reduce(f, l, i) → Any — Reduce list l with function f and initial value i
  • min(l) → Number — Minimum value in list l
  • max(l) → Number — Maximum value in list l
  • list_slice(l, from, to) → List — Slice list l from from to to (exclusive)
  • reverse(l) → List — Reverse the list l
  • zip(l1, l2) → List — Pair elements from l1 and l2 (shortest length)
  • zip_longest(l1, l2) → List — Pair elements from l1 and l2 (fill with none if lengths differ)
  • change_value(l, i, v) → List — Change value at index i in list l to v
  • sort(l, reverse) → List — Sort list l (reverse: 0=asc, 1=desc)
  • count(l, v) → Number — Count occurrences of v in l
  • index_of(l, v) → Number/none — Index of v in l or none if not found
  • clear_list(l) → none — Remove all elements from l
  • list_multiply(l, n) → List — Repeat list l n times
  • rand_int_list(len, min, max) → List — List of random integers
  • rand_float_list(len, min, max) → List — List of random floats

string.zer

  • split(s, sep) → List — Split string s by separator sep
  • str_slice(s, from, to) → String — Slice string s from from to to
  • join(sep, l) → String — Join list l with separator sep
  • replace(s, old, new) → String — Replace old with new in s
  • to_upper(s) → String — Uppercase s
  • to_lower(s) → String — Lowercase s
  • ord(s) → Number — Unicode code point of character s
  • chr(n) → String — Character from code n
  • is_digit(s) → Bool — Is s a digit
  • is_ascii_lowercase(s) → Bool — Is s a lowercase letter
  • is_ascii_uppercase(s) → Bool — Is s an uppercase letter
  • is_ascii_letter(s) → Bool — Is s a letter
  • is_space(s) → Bool — Is s a whitespace character
  • find(s, v) → Number/none — Index of substring v in s
  • find_all(s, v) → List — All indices of substring v in s
  • startswith(s, v) → Bool — s starts with v
  • endswith(s, v) → Bool — s ends with v
  • str_multiply(s, n) → String — Repeat string s n times

math.zer

  • sqrt(a) → Number — Square root of a
  • abs(a) → Number — Absolute value of a
  • fact(n) → Number — Factorial of n
  • sin(x) → Number — Sine of x
  • cos(x) → Number — Cosine of x
  • tan(x) → Number — Tangent of x
  • gcd(a, b) → Number — GCD of a and b
  • lcm(a, b) → Number — LCM of a and b
  • fib(n) → Number — n-th Fibonacci number
  • is_prime(n) → Bool — Is n prime
  • deg2rad(d) → Number — Degrees to radians
  • rad2deg(r) → Number — Radians to degrees
  • exp(x) → Number — Exponential of x
  • log(x) → Number — Natural logarithm of x
  • sinh(x) → Number — Hyperbolic sine
  • cosh(x) → Number — Hyperbolic cosine
  • tanh(x) → Number — Hyperbolic tangent
  • round(x) → Number — Round x to nearest integer
  • Constants: PI, E, ln2

file.zer

  • write(file, mode, text) → none — Write text to file with mode mode ("w" or "a")
  • read(file) → String — Read contents of file
  • exists(file) → Bool — Check if file exists
  • get_dir() → String — Get current directory
  • set_dir(dir) → none — Change current directory
  • list_dir(dir) → List — List files in directory
  • make_dir(dir) → none — Create directory
  • remove_file(file) → none — Remove file
  • rename(old, new) → none — Rename file or directory
  • remove_dir(dir) → none — Remove directory
  • copy(src, dst) → none — Copy file
  • is_file(file) → Bool — Is path a file

hash.zer

  • md5(s) → String — MD5 hash of s
  • sha1(s) → String — SHA1 hash of s
  • sha256(s) → String — SHA256 hash of s
  • sha512(s) → String — SHA512 hash of s
  • crc32(s) → String — CRC32 hash of s

memory.zer

  • remember(key, value) → none — Store value with key
  • recall(key) → value/none — Retrieve value by key
  • forget(key) → none — Remove key from memory
  • clear_memory() → none — Clear all memory
  • keys() → List — List all keys
  • is_empty() → Bool — Is memory empty
  • size() → Number — Number of keys

net.zer

  • get_ip() → String — Get public IP address
  • get_mac() → String — Get MAC address
  • ping(host) → String — Ping host
  • downl(url) → File — Download file from URL
  • get_local_ip() → String — Get local IP address
  • get_hostname() → String — Get hostname

random.zer

  • rand() → Float — Random float in [0, 1]
  • rand_int(min, max) → Int — Random integer in [min, max]
  • rand_float(min, max) → Float — Random float in [min, max]
  • rand_choice(list) → Any — Random element from list
  • int_seed(i) → Int — Integer seed
  • float_seed(f) → Int — Float seed

sys.zer

  • system(cmd) → none — Execute system command
  • get_env(name) → String — Get environment variable
  • set_env(name, value) → none — Set environment variable
  • exit(code) → none — Exit program
  • argv — List of command-line arguments
  • os_name — OS name

threading.zer

  • thread_start(func, args) → Thread — Start a new thread
  • thread_sleep(seconds) → none — Sleep current thread
  • thread_join(thread, timeout) → none — Wait for thread to finish
  • thread_is_alive(thread) → Bool — Is thread alive
  • thread_cancel(thread) → none — Cancel thread

time.zer

  • sleep(seconds) → none — Sleep for seconds
  • time() → Float — Current time (seconds since epoch)
  • ctime(time) → String — Human-readable time

keyboard.zer

  • keyboard_write(text) → none — Type text as keyboard input
  • keyboard_press(key) → none — Press a key
  • keyboard_release(key) → none — Release a key
  • keyboard_wait(key) → none — Wait for key press
  • keyboard_is_pressed(key) → Bool — Is key pressed

How to Use Libraries

  • To use a library, add load "libs.<name>" at the top of your .zer file.
  • Example: load "libs.math"
  • Then call functions as documented above.

Zerion Editor

About Zerion Editor

  • Modern, lightweight code editor designed for Zerion language
  • Syntax highlighting, auto-complete, file explorer, markdown preview
  • Open images, manage folders, and more
  • Cross-platform (Windows/Linux/Mac, requires Python 3.11+ and PyQt5)

Key Features

  • Tabbed editing, drag & drop file tree, and project explorer
  • Live Markdown preview for .md files
  • Quick commenting/uncommenting for Zerion code
  • Auto-completion for keywords, functions, and types
  • Image viewer for common formats (PNG, JPG, GIF, etc.)

Keyboard Shortcuts

  • Ctrl+N: New file
  • Ctrl+O: Open file
  • Ctrl+K: Open folder
  • Ctrl+Shift+K: Close folder
  • Ctrl+S: Save file
  • Ctrl+Shift+S: Save as
  • Ctrl+F: Find
  • Ctrl+H: Find and Replace
  • Ctrl+B: Toggle file tree
  • Ctrl+Shift+E: Show explorer
  • Ctrl+Q: Exit
  • Ctrl+/: Toggle comment (Zerion files)
  • Ctrl+P: Toggle Markdown preview
  • Ctrl+X: Cut line (when nothing selected)
  • Ctrl+Z / Ctrl+Y: Undo / Redo
  • Ctrl+A: Select all
To launch: python zerion_editor.py
